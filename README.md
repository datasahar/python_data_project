# 📌 Phone Book Project – Python

This project is a simple Phone Book application built using Python.

The program allows the user to enter a phone number, then searches for it in the phone book and returns the name of the person if the number exists.

If the number is not found, the following message will appear:

Sorry, the number is not found

If the entered number is invalid (less or more than 10 digits, or contains letters or symbols), the program will display:

This is invalid number


## 🔧 Program Features
- Search for a person's name using their phone number.
- Search for a phone number using a person's name.
- Add a new contact to the phone book.


## 🧠 Programming Concepts Used
- Functions
- Dictionary
- Using loops with dictionaries
- Using if statements inside loops
